# 텔로 class
텔로 sdk는 기본적으로 3가지 구조를 가진다.
1. 명령어를 보내고 응답을 받는 것
2. 두번째는 기기의 상태를 받는 것
3. 비디오 스트림을 받기

## 텔로 명령어 보내고 응답 받기
텔로를 wifi 1:1로 연결하고나면 명령어를 보낼 수 있다. 다음은 명령어 목록과 설명, 그리고 명령뒤에 텔로에서 보내오는 응답을 설명한다.

### 회신이 ok와 error인경우

* Command : command 명령은 텔로가 sdk에서 제공하는 명령을 받을 수 있도록 지시한다. 
* takeoff : 이륙
* land : 착륙
* streamon : 비디오 스트림 시작
* streamoff : 비디오 스트림 끝
* emergency : (위급상황)기체를 포기하고 즉시 모터를 끈다.
